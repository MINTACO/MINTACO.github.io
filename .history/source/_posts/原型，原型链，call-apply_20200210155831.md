# 1.原型
# 1.1 原型概念
原型是function对象的一个属性，它定义了构造函数制造出的对象的公共祖先。通过该构造函数产生的对象，可以继承该原型的属性和方法。原型也是对象。   

```
//利用原型特点，可以提取公有属性
// Car.prototype.carName = "BMW";
// Car.prototype.length = 5000;
Car.prototype = {
    carName : 'BMW',
    length : 5000
}
function Car(owner, color){
    this.owner = owner;
    this.color = color;
}
```
